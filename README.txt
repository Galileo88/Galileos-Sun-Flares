To Install:

1. Install Scatterer
2. Choose the color you would like your Sunflare to be
3. Choose which sunflare you want (1,2,3,4, Etc...)
4. Drop the "scatterer" folder into your GameData folder and overwrite everything when prompted.
5. Enjoy
